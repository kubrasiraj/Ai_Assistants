# AI LinkedIn Post Generator

This is a simple AI-powered project that generates professional LinkedIn posts using LangChain and Mistral AI.

I built this project while learning Generative AI and LangChain to improve my practical coding skills and understand how LLM-based applications work in real life.

---

## 💡 What this project does

- Takes input like topic, audience, and tone
- Uses an AI model (Mistral) to generate LinkedIn-style posts
- Produces human-like, engaging, and structured content
- Helps practice prompt engineering and LangChain basics

---

## 🛠️ Tech Stack

- Python 🐍  
- LangChain 🔗  
- Mistral AI 🤖  
- Streamlit (optional UI version)

---

## 📌 Features

- Simple and interactive prompt system  
- Human-like LinkedIn post generation  
- Customizable tone and audience  
- Beginner-friendly AI project

## 📚 What I learned

While building this project, I learned:

- How LLMs process prompts
- Basics of LangChain PromptTemplate
- How to structure inputs for better AI output
- How to connect AI models in real applications
